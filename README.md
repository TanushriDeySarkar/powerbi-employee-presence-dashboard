# powerbi-employee-presence-dashboard
Power BI dashboard analyzing employee attendance, WFH trends and sick leave insights.

## Project Overview
This Power BI dashboard analyzes employee attendance patterns including:
- Attendance %
- Work From Home (WFH) trends
- Sick Leave (SL) analysis

## Key Insights
- Overall attendance rate: 91.8%
- WFH percentage: 10%
- Sick leave percentage: 1.1%
- Monday has the highest attendance.
- Friday shows the highest WFH trend.

- Power BI
- Data Visualization
